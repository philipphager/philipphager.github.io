# Website

## Install
1. Install ruby and follow instructions: `brew install ruby`
2. Setup [jekyll](https://jekyllrb.com/): `gem install bundler jekyll`
3. Install dependencies: `bundle install`
4. Run: `bundle exec jekyll serve`




[Oinam Jekyll](https://rubygems.org/gems/oinam-jekyll)

```
$ gem build oinam-jekyll.gemspec
$ gem push oinam-jekyll-x.x.x.gem
```
